# Archive Scripts

This directory contains deprecated utility scripts that were used during the initial setup and organization of the MinDatabase repository.

## Contents

### add_sections.py

A Python utility script used to add section headings to artist biography markdown files. This script was designed to insert markdown headers (##) into prose texts that lacked formal section markers.

**Purpose**: One-time restructuring tool to add consistent section structure to artist biographies.

**Status**: Deprecated - The restructuring work has been completed and this script is retained for historical reference only.

**Original Location**: `Content/Artists/XIII century/add_sections.py`

## Note

These scripts are archived for documentation purposes and are not intended for active use. Active maintenance scripts should be placed in `scripts/maintenance/`.
